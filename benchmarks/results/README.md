# Benchmark Results

Committed benchmark results should be added here after running a reproducible profile on a named machine configuration.

Generated local reports are written under `artifacts/benchmarks/<profile>/reports/` and are intentionally ignored by git. Copy only curated result summaries into this folder, including:

- FileSearch commit SHA
- profile name
- CPU, memory, disk, OS, and power mode
- command used
- report JSON or Markdown summary

Do not publish competitor performance claims unless the comparison corpus, hardware, versions, and settings are documented here.
