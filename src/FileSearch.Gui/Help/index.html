FileSearch Help

Find text across files quickly and confidently.

FileSearch searches a folder for files containing text, regular expressions, or Boolean expressions. It can scan source files, text documents, Office files, PDFs, other common document formats, and opt-in OCR text from images, scanned PDFs, and embedded document images.

Choose a scope

Select the folder in Look in, decide whether subfolders are included, then optionally narrow the scan with file name patterns like *.cs;*.xaml.

Use Quick Search

Press Win+Shift+F to open a small search window from anywhere. It searches indexed locations, streams results, and closes after you open, reveal, copy, pin, or drag a result.

Launch the release

Run the installed desktop app for interactive search, use the CLI executable for scripts, or let FileSearch manage the indexer and extractor helper processes.

Use the CLI

Run FileSearch.Cli.exe from the release folder for one-shot searches, structured output, index commands, workflows, and indexer status.

Pick a query mode

Use Boolean mode for normal search expressions, Plain text for literal phrases, or Regex when you need a .NET regular expression.

Review matching files

The results grid shows every matched file, hit count, first matching line, and folder. Selecting a row opens the Hits tab with matching lines and context.

Fast Start

1

Enter what you want to find

Type the text or expression into Containing text. Press Enter or choose Start to search.

2

Limit the file set

Use File name pattern for globs such as *.cs, *.txt, or *.cs;*.xaml;*.md. Leave it blank to search every supported file.

3

Choose the folder

Enter a path in Look in or use the folder button. Turn Include subfolders on when you want recursive search.

4

Inspect and refine

Select a result to see matching lines. Open the Filter tab to narrow the current results without scanning the files again.

Boolean search is the default. Adjacent words are treated like AND, so error timeout means files must contain both terms.

Common Tasks

Search only source files

Set File name pattern to something like *.cs;*.xaml;*.json, then search from your solution folder with subfolders enabled.

Find TODO or FIXME comments

Use Regex mode with TODO|FIXME, or Boolean mode with TODO OR FIXME.

Search an exact phrase

Use Plain text mode, or wrap the phrase in quotes in Boolean mode: "connection string".

Search from Explorer

Use File > Install Windows integration to add a FileSearch command for folders and folder backgrounds.

Speed up repeat searches

Add folders with Index > Manage indexed locations, then enable Use index. The GUI or tray indexer can keep them current in the background. See Indexed Search.

Search all indexed locations

Use Quick Search with All indexed locations when you do not remember which root contains a file.

Reuse a file pattern

Use New scope in the sidebar to save a named pattern such as *.cs;*.xaml, then apply it with one click before searching.

Prefer a denser layout

Use View > Style > Compact or File > Settings > Layout style to reduce spacing in controls, lists, and toolbars.
