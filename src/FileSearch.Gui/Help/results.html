Results

Review, refine, and open matching files.

Search results are grouped by file. Select a file to see matching lines, use the Filter tab to narrow the current result set, or open files directly from the grid.

Results Grid

Column Description
Name The matched file name.
Hits The number of matching lines or spans found in that file.
First match A short preview of the first matching line.
Location The folder containing the file.

Hits Tab

Selecting a file automatically opens the Hits tab. The preview shows matching line numbers and nearby context. For large or binary-backed document formats, the preview may be simplified depending on the extractor.

Summary Tab

The Summary tab shows search statistics: files matched, total hits, and elapsed time. The status bar gives additional live progress while a scan is running.

Filter Tab

The Filter tab narrows the current results in memory. It matches against file names and hit lines, so it is useful after a broad search.

  • Filtering does not re-open or rescan files.
  • Clear the filter field to show all results from the search again.
  • Run a new search when you need to change folder, file pattern, query mode, or date and size filters.

Opening and Copying

Double-click a result row to open the file. Right-click a row for more actions:

  • Open opens the file with the default Windows application.
  • Reveal in Explorer opens the containing folder and selects the file.
  • Copy file path copies the full path to the clipboard.
  • Copy folder path copies only the containing folder path.
