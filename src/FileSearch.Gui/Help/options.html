Options

Tune scope, speed, and matching behavior.

Most searches only need the Main tab. Use Options and Dates when you want stricter query behavior, faster scanning, or file metadata filters.

Query Options

Option Behavior
Search mode Controls whether the query is parsed as Boolean, Regex, or Plain text.
Match case When enabled, uppercase and lowercase letters must match exactly.

Document Extraction

Option Behavior When to use it
Office & PDF documents When enabled, FileSearch extracts text from supported Office and PDF documents. When disabled, those document types are skipped. Disable it for faster source-code-only searches.
Image OCR When enabled, FileSearch uses Windows OCR to search recognized text in PNG, JPEG, BMP, TIFF, scanned PDF pages, and embedded images in Office, OpenDocument, EPUB, email, and ZIP/archive files. Enable it for screenshots, scanned image files, scanned PDFs, image-heavy reference folders, and documents that contain screenshots or scanned attachments. Leave it off for faster searches through photo or asset folders. OCR language and scanned-PDF page limits are configured in File > Settings.
Skip unknown file types When enabled, FileSearch only scans known text/document extensions and any additional extensions you provide. Enable it when a folder contains many binary or generated files.
Additional plain-text extensions Adds custom text-like extensions such as .liquid, .tmpl, or .razor. Use it with Skip unknown file types when your project has uncommon text files.
For scanned or image-only PDFs, Office & PDF documents and Image OCR both need to be enabled. A file name pattern such as *.pdf only limits the files being searched; it does not enable OCR.

File Size Filters

Minimum and maximum file sizes are entered in kilobytes.

  • Min excludes files smaller than the entered size.
  • Max excludes files larger than the entered size.
  • A maximum of 0 means no upper size limit.

Date Filters

The Dates tab filters by modified date. Use After, Before, or both to search only files changed in a specific period.

The Before date is inclusive for the whole selected day, so a file modified at 11:59 PM on that date still qualifies.

Indexed Search

The Use index checkbox next to the search controls lets covered searches read from the local file index instead of rescanning files; live scan remains the fallback when a search is not covered. Indexed locations, background indexing, pause/resume, health, and status badges are described on the Indexed Search page.

Background Indexer

Use File > Settings to control whether FileSearch stays resident after the search window closes and whether the tray indexer starts when Windows starts.

  • Keep index updated after closing window hides the main window and keeps FileSearch running in the tray, so indexing and Quick Search stay available.
  • Start background indexer when Windows starts starts the tray indexer after user sign-in. This is not a Windows Service.
  • Pause indexing on battery, Indexer resource use, CPU limit, and Disk pause reduce background indexing impact.

Quick Search Settings

Use File > Settings > Quick Search to choose the global hotkey, default scope, whether the last scope is remembered, which indexed roots are included by the Selected indexed locations scope, and the shortcuts used inside the quick window.

  • Global hotkey can be Win+Shift+F, Alt+Space, or Ctrl+Space.
  • Default scope is used when remembering the last Quick Search scope is off.
  • Include indexed content controls whether Quick Search adds indexed file-content matches after filename and path matches.
  • Selected indexed locations controls which roots are searched by that quick scope.
  • Quick window shortcuts controls close, focus results, open, reveal, copy path, pin, and preview actions inside Quick Search.

Themes

Use View > Theme to choose Light, Dark, Visual Studio, or System. Theme selection is saved and restored the next time FileSearch starts.

Windows Integration

Use File > Install Windows integration to add FileSearch to the Windows folder context menu and Start menu. Use Remove Windows integration to remove those registrations.
